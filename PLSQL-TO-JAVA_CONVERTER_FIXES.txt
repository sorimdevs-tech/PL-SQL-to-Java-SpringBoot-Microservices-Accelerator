PRIMARY FINDING
═════════════════════════════════════════════════════════════════════════════

The LLM is BYPASSED for regular packages. Generator only uses deterministic
Python templates that build Java by string concatenation, with NO AI validation.

  generate_services_from_semantics()
    ├─ _generate_utility_service_from_unit()        [No SQL]

       ✗ Never validates against Oracle syntax detection
       ✗ Produces code with NVL, INSTR, SUBSTR, APEX, etc.
